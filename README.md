### How to run
First of all, you have to clone the repository. Either download the main branch as .zip:
![Download branch as zip](https://i.imgur.com/GC1wa5d.png)
...Or use git:
```
git clone https://github.com/CuteBladeYT/Everest
```

<br>

Then you can cd into the directory using
```
cd Everest
```
And install all the needed dependencies.
```
npm i
```
And if the job is done successfully,
```
npm start
```
